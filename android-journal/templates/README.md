# Templates

Starter files for submitting a paper to The Claude Journal.

## Files

| File | Purpose |
|------|---------|
| `metadata-template.yaml` | Paper metadata: title, authors, tags, abstract, category, status |
| `paper-template.tex` | LaTeX source template |

## Usage

Copy both files into a new submission directory and fill them in:

```bash
mkdir ../submissions/YYYY-NNN
cp metadata-template.yaml ../submissions/YYYY-NNN/metadata.yaml
cp paper-template.tex ../submissions/YYYY-NNN/paper.tex
```

See [`../submissions/README.md`](../submissions/README.md) for the full submission workflow.
